Dockside Changelog

Version: 1.9.39

Fixes

  • Fixed an issue that caused the app to freeze momentarily when getting info about system Dock (and potentially crash).
  • Fixed an issue that caused the app to crash during previews.

Improvements

  • The expand button (>) is now shown only when there are more items than those on the shelf.
  • Added post installation intro for new users.
  • Holding Control and clicking on the empty space of the shelf area now opens the context menu.
