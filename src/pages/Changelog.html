Dockside Changelog

Version: 1.8.6

Improvements

  • Worked on optimising the 'Enhanced Sizing Response' option to consume less CPU when polling for Dock size changes.
  • Improvements to the Dockside auto-sizing based on the activity in Dock.
  • Added support for dragging/downloading remote server file via Visual Studio Code app (support for other apps coming soon, reach out to add your favourite apps via email: hachipoo.apps@gmail.com)
  • Dragging detection now ignores a few more non-relevant items.
  • Tuned the full screen detection to account for animations/transitions delays in few apps.
  • The Finder Sync Extension (that provisions the Finder right-click menu item "Add to Dockside") is restarted correctly with Dockside app launch.
