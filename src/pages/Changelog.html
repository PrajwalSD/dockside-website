Dockside Changelog

Version: 1.9.6

New Features

  • Option to disable menu bar popups is added (useful when you hide menu bar icons using third-party apps like Ice, as the Dockside popups can be annoying and misplaced)

Improvements

  • Further improvements to Dockside reveal animation in auto-hide mode based on user feedback.
  • Introduced a one-time notification when deleting an item for the first time in the Screenshots and Downloads shelf.
  • Option to hide menu bar icon is now added.
