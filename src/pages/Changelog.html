Dockside Changelog

Version: 2.9.0

New Features

  • You can now set auto-dismiss timer for the drop shelf.
  • For better legibility on the hovered file, the neighbouring icons in the shelf now auto-dims, you can turn this off it's too jarring in the advanced settings.
  • You can now increase the font size for the filename labels in settings
  • You can now adjust space between icons in the shelf, for comfortable layouts.
  • Search in settings

Fixes

  • Further refinements to the Liquid Glass implementation (obsessed with this!)
  • Fixed the folder monitoring issue with drop shelf
