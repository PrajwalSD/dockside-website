Dockside Changelog

Version: 1.9.1

New Features

  • Added "Restart Dockside..." option.
  • New hotkey - when dragging items to the Files shelf, you can now temporarily copy the item to the shelf instead of storing it as a reference by pressing and holding the Option key during the drag.

Improvements

  • Fixed flickering icon bug.
  • Improvements to Dockside performance, should feel more lighter and responsive to use.
