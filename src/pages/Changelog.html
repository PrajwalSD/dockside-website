Dockside Changelog

Version: 1.9.4

New Features

  • Added "Send via Email..." option to the file menu.

Improvements

  • Improvements to the automatic updates check.
  • Compressing of large files are pushed to the background now.
  • Improvements to the file previews, added support for few more image and video formats.
  • File size optimization (using Clop) now occurs asynchronously.
  • General improvements to the handling of zero-byte / iCloud placeholder items in Dockside.
  • Graceful handling and indication of cloud placeholder (zero-byte) files on the shelf.
