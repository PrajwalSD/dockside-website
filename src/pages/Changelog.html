Dockside Changelog

Version: 1.9.29

New Features

  • When dragging an item to the shelf, you can configure Dockside to actually move the original item to the shelf. Configure this in "Advanced" settings.
  • Added a toggle to completely disable Screenshots & Downloads shelf.

Fixes

  • Fixed a bug that disabled Dockside auto-hide mode when switching between screens.
  • Fixed a bug that caused app to crash on right-click a shelf item.

Improvements

  • Minor updates to the Settings menu
  • Further minor refinements to the drag to folder feature.
