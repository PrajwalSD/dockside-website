Dockside Changelog

Version: 1.9.14

New Features

  • You can now trigger a Quick Look preview using a keyboard shortcut - hit space bar when you click an icon in the shelf to reveal a full sized Quick Look preview.

Fixes

  • Reduced the sensitivity level of text/string drag detection in other text editor apps.
  • Added "HazeOver" to the list of apps to be ignored from potentially triggering Dockside to go into auto-hide mode during system restarts.
  • Fixed an issue where the Dockside Settings panel (if open) would aggressively reset the space.
  • Fixed an issue where the filename in the quick preview appeared in black text in dark mode, making it illegible.
  • Fixed broken open -a command

Improvements

  • Mouse-click will now show quick preview too.
  • Implemented handling of custom URL schemes (`dockside://`) for app actions to support third-party apps to better integrate with Dockside and for users to aid with scripting/automation. See https://hachipoo.com/integrations for more info.
