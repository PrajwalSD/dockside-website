Dockside Changelog

Version: 2.6.1

New Features

  • Added gesture configuration for each shelf, you can now customise mouse gestures for each shelf in the settings.

Fixes

  • Better full screen app management - revised implementation that works seamlessly across spaces & screens.
  • Fixed issues with right-click context menu when multiple sources are configured in the area.
  • Fixed colours to align with native colour palette of macOS
  • Removed count limits, since it is automagically managed.
  • Added support for non-iCloud files (triggers auto-download on adding to shelf etc.)
