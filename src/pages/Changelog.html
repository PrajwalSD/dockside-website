Dockside Changelog

Version: 2.9.19

New Features

  • You can now order the categories in the drop shelf (move them up/down)
  • Improvements to legibility issues with transparent mode especially when expanding over the Dock.

Fixes

  • Fixed a bug that reset the light/dark theme setting when toggling between transparent <> dock style modes.
  • Fixed an issue where shelf style was reset on Dock resizing specifically with transparent mode enabled.
  • Fixed issue with folder drag out (was not working)
