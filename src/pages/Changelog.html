Dockside Changelog

Version: 2.9.5

New Features

  • Grid shelf, if it results in a file/object, a drop shelf will automatically popup to present it as a convenience.
  • Extended the screenshots monitoring in Clipboard, support third-party tools and custom keyboard shortcuts.
  • You can now control the offset from screen edge for shelves
  • Additional grid shelf cell types - scripts, services, quick action etc.

Fixes

  • Fixes an issue with single shelf mode, didn't respect the Dock's existence on the same side of the screen.
  • Fixes an issue with settings panel not respecting small screen sizes
  • Potential fix for an issue with iMessage drop zone.
  • App icon in Dock (option in settings)
  • Drop shelf navigations are a bit better with grid mode explicitly stated.
  • Fixed an issue with cycle through sources functionality
  • General bug fixes and improvements to the core modules
