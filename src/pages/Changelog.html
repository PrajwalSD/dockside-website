Dockside Changelog

Version: 2.8.8

New Features

  • Added two new Liquid Glass variants that match the Dock (check out Dock (Dark) and Dock (Light) variants).
  • Extended the email drop zone to include advanced options (experimental)
  • Added live tuning sliders for Apple Dock glass filter params in preferences
  • Override CAFilter params for Apple Dock variant to match real Dock.app glass appearance
  • Added Services integration

Fixes

  • Fixed Apple Dock variant to use style=.clear which triggers the correct transparent blur glass internally (_variant=2), matching the actual Dock.app appearance.
  • Added "Apple Dock" Liquid Glass variant that matches Apple's Dock.app exactly using _subvariant="dock". [RN] Fixed NSGlassEffectView private property names (_scrimState, _subduedState) that were silently failing.
  • Fixed the issue where scrollbar was cutting off cards in the list view (page) mode.
  • The main shelves will show some visual cue instead of displaying an empty shelf in smaller width setting.
