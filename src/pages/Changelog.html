Dockside Changelog

Version: 2.9.7

New Features

  • Redesigned settings page! Share your opinion.
  • Added "Reduce Motion" control to limit animations.
  • You can now configure left/top shelf to be a media or stats shelf (magic!)
  • Exclude folders, external drives, cloud drives from folder navigation in shelf. See advanced settings to add exclusions.
  • You can now set one of the area in right/bottom shelf as media or info/stats type.
  • You can now adjust the offset/gap away from the Dock and away from the edge of the screen for super customised positioning of the Dockside shelves!

Fixes

  • Fixed an issue that triggered pagination expansion mode when deleting or removing the file from the shelf.
  • Fixes an issue where downloads area was excluding folders.
  • Fixes an issue that reset the shelf material on docking with external monitors.
  • Fixed dragging issue with Finder getting into a wrong state.

  • Fixed an issue with drop shelves duplicating the clipboard content (e.g., screenshots) when clipboard history is enabled in the right/bottom shelf
  • Fixes to the near cursor shelf behaviours and glitches.
  • Core performance improvements and general bug fixes
  • To make the row cards legible in white/light mode, the list view cards are now rendered in dark mode.
  • Fixed favicon related issues in Clipboard History.
  • Fixed minor glitch in text marquee in drop shelf.
  • Fixed wider right-click context menu issue.
