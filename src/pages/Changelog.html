Dockside Changelog

Version: 1.9.53

New Features

  • Dockside now supports Shortcuts integration with useful actions for adding files, retrieving shelf contents, clearing files, and restoring deleted items.

Fixes

  • Fixed an issue with dragging out items from shelf.
  • Fixed a bug that was making the icon non-responsive to the mouse hover.
  • Fixed an annoying bug with auto-expand shelf feature.
  • Resolved an issue where Dock autohide delay settings were not properly inherited by Dockside settings.
  • Files on shelf didn't respond to the mouse hover at times (especially post resizing of the shelf), this is handled better now and it should avoid any glitches around this.

Improvements

  • Added a one-time warning when removing or deleting a file from the shelf.
