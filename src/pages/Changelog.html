Dockside Changelog

Version: 2.9.13

New Features

  • New browser/system media playback control support in the media mode (experimental)
  • New keyboard shortcut, select > cmd+backspace removes or deletes the file depending on the shelf.

Fixes

  • Bug fix related to start up.
  • The categories selection is persisted now in shelf manager
  • The empty shelf manager is not auto-closed.
  • Renamed/saved drop shelves are now only shown once in the recents list.
