Dockside Changelog

Version: 2.2.0

New Features

  • Redesigned onboarding flow and launch screens.
  • Clipboard monitoring in shelf - now you can use the Screenshots shelf to drop clipboard items to shelf.
  • Introducing colour tags and filters in shelf! (love this idea, came from a user who's been a big contributor in some of the recent features!)
  • Trigger Quick Look preview on single click for files in the shelf (you can also hit the Space bar when in preview to trigger Quick Look as well!). Double-click top open the file using default app.

Fixes

  • General improvements and bug fixes.
  • Critical additional trial and licensing related security fixes.
  • Fixed the drag area drop zone overlap issue.
  • Ability to duplicate the selected files in the shelf
