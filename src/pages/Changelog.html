Dockside Changelog

Version: 2.8.7

New Features

  • Pagination expansion is now always enabled in list view mode.
  • You can add apps to ignore list for drop shelf, if they annoy your workflow.

Fixes

  • Fixed issues related to mouse gestures - open/closing of the shelf in list view mode.
  • When a new drop shelf is already shown, continued stale shake to drop gestures are ignored now.
  • General improvements and bug fixes
  • Copy/Move to a path pop up has fixed now.
  • Potential fix for file preview lifecycle - users have complained that at times the file previews don't show up, requires a refresh or restart to make it work again.
