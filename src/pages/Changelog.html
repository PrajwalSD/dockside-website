Dockside Changelog

Version: 1.9.21

New Features

  • Added enhanced customisation options for Dockside's show/hide mode. Users can now choose to reveal a specific shelf upon mouse hover when in auto-hide mode. This was added based on user feedback.

Fixes

  • Addressed the stuttering window issue triggered when using the Shift+Fn keyboard shortcut to temporarily reveal the Dockside shelves. This might still be an issue in newer macOS, will address this properly in subsequent releases.

Improvements

  • General bug fixes and refinements
  • Further refinements to the Dockside show/hide sensitivity on mouse hover.
  • Minor UX update to the selection checkbox toggle behaviour
