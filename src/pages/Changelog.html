Dockside Changelog

Version: 2.9.4

New Features

  • Ability to cycle through the data sources in the shelf - a new way to see all of your files in a single shelf you can now choose to seem them all in one go or cycle through the sources.
  • Added the ability to rename the stacks on the shelf
  • Clipboard history can now be cleared via right-click options on the shelf
  • Added a new permanent behaviour in setting called "Move Files on Dragging Out" - works like the "Cmd" hotkey but can now be toggled permanently for the shelves including drop shelf.
  • Added zip option for email drop zone + send via email right click action will now also show the drop zone action if it's configured.
  • Added drop zone actions in the right click menu
  • Added the ability to customise the corner radius of the drop shelf ;)

Fixes

  • Dragging attachments/content from Outlook should work now.
  • Fixed folder drag out issue
  • Fixed right-click issue with folders in the shelves.
  • Error messages are better handled to ensure the user is informed of the errors, specifically when configuring complex workflows using drop zones
  • Fixed pagination expansion issue (in list mode) where the multiple areas were not handled.
  • Fixed freezing after pagination expansion in some scenarios
  • Bug fixes to drop shelf
  • Refinements to the Drop shelf (numerous bug fixes and refinements included)
