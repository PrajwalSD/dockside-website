Dockside Changelog

Version: 1.9.52

New Features

  • Adding support for thumbnail preview in shelves!

Fixes

  • The action buttons were not activating on mouse hover, fixed this now.
  • Potential fix for an issue which caused settings panel to launch behind other open app windows!
  • A bug related to full-screen application detection at launch has been resolved.
  • Floating thumbnail for screenshots system option was not being read correctly (in Dockside settings), fixed this.
  • Fixed unhandled errors related to downloads shelf.
  • Recent files are now properly ordered and refresh button should be working as expected.

Improvements

  • Fade in/out is disabled in auto-hide mode as it is rather redundant.
  • Increased the default item count in shelves.
  • Adjusted the animation delay to further sync with customised Dock delay animations.
