Dockside Changelog

Version: 1.9.15

Fixes

  • Fixed .csv Quick Look preview issue.
  • Fixed the filename label colour issue to match the system.

Improvements

  • In left/right Dock position Dockside now takes most of the available screen space at the bottom of the screen.
  • The show (full screen) mode in Microsoft PowerPoint is now detected and Dockside will show/hide accordingly.
  • Increased item label font size for better legibility.
  • Improvements to performance of Dockside in auto-hide mode.
