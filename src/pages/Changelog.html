Dockside Changelog

Version: 1.8.5

Improvements

  • [N] Added a FinderSync extension to the app, enabling users to add items to Dockside directly from Finder using the right-click menu.
  • Extended the fluid animations when presenting quick action buttons on icons.
  • When an app from the ignore/exception list enters full-screen mode, Dockside will show a one-time warning explaining why auto-hide is disabled. This minor change helps users identify and address any accidental additions to the ignore list.
  • Removed the redundant "Set Dockside Screen" option in the right-click menu when no external monitors are connected and same with the Settings menu.
