Dockside Changelog

Version: 1.9.36

New Features

  • Added a new option to toggle icon decorators (or visual indicators such as green dot, bolt icon etc.) in settings.
  • Drop area activation can now be customised to suit your needs (see Files shelf settings). You can also configure the shelf to appear near the mouse cursor for easier drag-and-drop.

Fixes

  • Fixed icon freeze issue seen when opening a file on the shelf.
  • Fixed an issue that caused re-activation of the notes area in the Files shelf in certain scenarios.
  • Fixed an issue where the shelf was not activating on drag event on the currently active screen, when in auto-hide mode.

Improvements

  • Further refinements to match the Dock show/hide timing.
  • The info banners that pop up are less annoying now.
