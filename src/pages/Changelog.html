Dockside Changelog

Version: 2.5.1

New Features

  • Introducing AI workflows in the shelf–only if you need to, by default this is off and you are required to provide you API key for the AI based workflows to work.
  • Drop zones has more actions + auto-trigger and much more! Drag areas now support drop zone wiring too.

Fixes

  • Pixel perfect icons on the shelf (empty state icons).
  • Complete control over configuring the areas to be in the right or bottom (screenshots and downloads) shelf + you can easily swap the areas too.
  • General improvements and bug fixes.
  • Minor UI uplifts to the shelf icons and text.
