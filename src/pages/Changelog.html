Dockside Changelog

Version: 1.9.56

New Features

  • Left/right Dockside placement! (you asked for it) You can now position Dockside on either side of the screen.
  • File stacks. Dragging multiple files onto the shelf keeps them in a single stack; you can revert to the previous behaviour in settings anyway.
  • Manual file ordering (you asked for it) in the Files shelf—drag and drop to rearrange.
  • Shelf colours (you asked for it) You can now set custom colours for the shelf—a much-requested feature.
  • Redesigned preferences and settings panels for a clearer, more intuitive experience.
  • Quick action popover for bulk actions.
  • Custom Notes location. Choose where Dockside stores the notes it creates.
  • Adjustable preview size (you asked for it)—enlarge or reduce the preview to suit your workflow.
  • Multi-language support (you asked for it): Spanish, Japanese, Chinese, and German. Localisation files are open source on GitHub for community contributions. Report translation issues to hachipoo.apps@gmail.com.

Fixes

  • Cloud sync. When you add a file to the shelf, Dockside automatically downloads it from the cloud if it isn’t available locally, so follow-up actions work seamlessly.
  • Animation and preview performance improvements to reduce glitches.
  • Bug fixes and refinements for compatibility with recent macOS Tahoe updates.

Improvements

  • Visual polish and stability refinements throughout. Liquid Glass style is still in the works!

Report issues

  • This is a major bump to Dockside, with many changes and bug fixes, if you face any issues, please report bugs at the earliest.
