Dockside Changelog

Version: 2.9.6

New Features

  • Added support for Audible in media mode on the shelf + you can select any other apps as long as they support and accept media control actions.
  • Clipboard History can now retain file references instead of copying over by default for local files, you can toggle this off in settings.

Fixes

  • App drawer has been restored after liquid glass implementation with subtle UX improvements.
  • Drag out issues leading to crash - potential fix has been applied.
  • Apps drags were rejected in the files shelf - fixed.
  • Favicon related fixes
  • General performance improvements and minor UI updates.
  • Clipboard history feature should work now, major regressions fixed.
