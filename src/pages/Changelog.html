Dockside Changelog

Version: 2.8.4

New Features

  • Settings for the Drop shelf has been improved and extended.
  • Hide shelf buttons when inactive for that zen/clean look + new settings button on the shelf.
  • Finder alternatives can now be added/configured for use in the Dockside
  • Double-click on the Drop shelf edge to dock the shelf to that edge of the screen + Move to and Copy to...features folder navigation now in Drop shelf

Fixes

  • Drop shelf right-click actions have been cleaned up.
  • Refinements and bug fixes to the recently released Drop shelf features and Liquid Glass implementations based on user feedbacks.
