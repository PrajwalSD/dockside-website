Dockside Changelog

Version: 2.9.2

New Features

  • Introducing Profiles in Dockside - you can now create multiple profiles with specific Dockside configuration to suit your workflow. Auto-trigger is quite basic at the moment but more will come.
  • Filename opacity, shadow can now be set for better legibility controls. Find them in Settings > Advanced > Shelf Elements.
  • Added reset options for few settings to avoid confusion.

Fixes

  • Fixed the in-progress indicator on the icon - more prominent spinner now.
  • Drop shelves don't popup on partly downloaded files (when background monitoring is enabled)
  • Count down visual cue is made simpler for auto-dismissing drop shelf
  • The auto-dismiss countdown timer is now red instead of accent colour to grab attention
