Dockside Changelog

Version: 1.9.44

New Features

  • Additional options to control the Dock auto-hide delay settings. Dockside inherits the Dock auto-hide delay settings already but now you have further control to fine tune the auto-hide animation.
  • Option to toggle Dock auto-hide delay. Dockside will automatically read the current Dock settings from system and inherit the timings to sync the show/hide behaviour with Dock, you can also further tune the Dockside timing with the new timing offset slider.

Fixes

  • Minor fix to the folder preview layout.
  • Fixed an issue related to shelf persisting in auto-hide mode.
  • Fixed an theme switch issue when Dockside is hidden.
  • Fixed the menu bar icon appearance now.

Improvements

  • Added support for the newly released Dropshare 6.
  • Further improvements and refinements to the Dockside auto-hide mode.
