Dockside Changelog

Version: 1.9.0

New Features

  • Introducing the App Drawer in Dockside – when you drag and drop apps into the shelf, Dockside will now automatically store them in an App Drawer for quick access, keeping the Files Shelf separate and focused on your files!
  • Popclip extension for Dockside is now available to all.

Fixes

  • Fixed icon flickers.
  • Fixed a bug where a note file appeared even when there wasn’t enough space to fit a single icon in the files shelf.

Improvements

  • New keyboard shortcut - create a new note in Dockside using ⌘N
  • Settings panel is now quick and responsive, as it should be.
  • Updated image and video formats accepted by Clop.
