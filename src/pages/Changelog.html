Dockside Changelog

Version: 1.9.19

Fixes

  • Fixed a bug that offloaded the items in the shelf on app launch.
  • Fixed an issue that didn't let users rename the file correctly.

Improvements

  • Minor improvement to system events permission alert - user can now reset the permission to re-trigger the macOS system events permission in the case it's not working as expected.
  • Dropshare generated URL is now available within Dockside for copy operation (rather than uploading again!)
  • Minor improvements to the overall UX
