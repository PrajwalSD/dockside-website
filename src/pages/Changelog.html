Dockside Changelog

Version: 1.8.7

New Features

  • You can now press and hold 'function' key when dragging to avoid Dockside from becoming active.
  • Added the ability to undo file removals from shelf. Use the "Restore Previously Removed Items" option in the Files Shelf to recover recently deleted files.
  • You can now adjust/delay the drag detection using advanced settings to suit your workflow and needs.

Improvements

  • The "Add to Dockside" (Services) option now works with plain text. Dockside extension submitted for Popclip app.
