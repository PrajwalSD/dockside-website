Dockside Changelog

Version: 1.9.26

New Features

  • All new (and useful) folder previews
  • You can now drop files into folders on the shelf by simply dragging items to the folder while holding the Function key. Similarly, you can drag files that are already in the shelf to the folders in the shelf. Press 'Command' key to toggle move instead of copy.

Improvements

  • Further refined the activation triggers for dragging
  • HotKey to reveal shelves temporarily has been changed to `Control + Option + Command` keys (instead of Shift + Function keys). Please note this.
  • The Dockside auto-hide mode toggle will now sync with the Dock auto-hide mode to reduce confusion.
