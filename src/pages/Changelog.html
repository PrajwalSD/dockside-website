Dockside Changelog

Version: 1.9.25

Fixes

  • Fixed an issue that prevented Dockside shelves from appearing on one of the displays in a multi-monitor setup.
  • Fixed an issue that caused Dockside shelves to retract too quickly.
  • Exclusions for recent files should now work as expected.

Improvements

  • In automatic positioning mode, the Dockside Files shelf will appear on the active screen (the screen where the mouse is located) when dragging is detected.
  • Minor bug fixes and general improvements to the app
